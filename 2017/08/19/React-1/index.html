Learn once and write anywhere

一套新的武功，都要有個聳動吸引人的標題。

這系列主要是我學習 React Native 的記錄

新接觸的語言、新的開發環境、新的設計思維

全新的挑戰！


前言

想想自己一開始是個 Andorid Developer，因為前公司人力短缺，被叫去支援開發 iOS，就這樣一路開發到現在，也因此愛上 iOS & Swift 。現在也因為公司的緣故，必須使用 React Native 來開發 Mobile App，現在就從跟 UI 有關係的 React 開始學習吧，藉此熟悉 Javascript + JSX，讓以後學習的道路更順遂一點，所以～我來囉！！


基礎知識

  • React 生態系 (ReactJS、JSX…等。 真的是學無止盡的汪洋……)

  • Expo 掃 Project 的 QRCode 就能 build App 的好東西，且不需要一直接線，只要在同一個網路環境底下即可。

  • NPM (Node Package Manager) 是管理第三方套件的一個工具，相當於 iOS 的 Cocoapods，Android 的 Gradle。(Note: 目前 React Native 還不支援 5 以上)
